# Extract list of desired graph features (IRFs anc GIRFs)

Extract list of desired graph features (IRFs anc GIRFs)

## Usage

``` r
Wished_Graphs_IRFandGIRF(InputsForOutputs, OutType, ModelType)
```

## Arguments

- InputsForOutputs:

  List of inputs for outputs

- OutType:

  Output types "IRF", "GIRF" and "IRF Ortho"

- ModelType:

  desired model type
