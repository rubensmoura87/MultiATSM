# Transformation of the JLL-related parameters (true form)

Transformation of the JLL-related parameters (true form)

## Usage

``` r
True_JLLstruct(ParaValue, Const_Type, FactorLabels, Economies, JLLinputs)
```

## Arguments

- ParaValue:

  Constrained parameter value

- Const_Type:

  Type of constraint

- FactorLabels:

  string-list based which contains the labels of all the variables
  present in the model

- Economies:

  string-vector containing the names of the economies which are part of
  the economic system

- JLLinputs:

  Inputs used in the estimation of the JLL-based models
