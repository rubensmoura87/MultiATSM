# Build country-specific link matrices

Build country-specific link matrices

## Usage

``` r
BuildLinkMat(GVARinputs, N, M)
```

## Arguments

- GVARinputs:

  List of inputs for GVAR-based models

- N:

  number of country-specific spanned factors (scalar)

- M:

  number of country-specific unspanned factors (scalar)
