# Extract list of desired graph features (IRFs anc GIRFs)

Extract list of desired graph features (IRFs anc GIRFs)

## Usage

``` r
Wished_Graphs_FEVDandGFEVD(InputsForOutputs, OutType, ModelType)
```

## Arguments

- InputsForOutputs:

  List of inputs for outputs

- OutType:

  Output types "FEVD", "GFEVD" and "FEVD Ortho"

- ModelType:

  desired model type
