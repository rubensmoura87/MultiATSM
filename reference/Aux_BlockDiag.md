# Transformation of the block diagonal parameters (auxiliary form)

Transformation of the block diagonal parameters (auxiliary form)

## Usage

``` r
Aux_BlockDiag(ParaValue, Const_Type, FactorLabels, Economies)
```

## Arguments

- ParaValue:

  Constrained parameter value

- Const_Type:

  Type of constraint

- FactorLabels:

  string-list based which contains the labels of all the variables
  present in the model

- Economies:

  string-vector containing the names of the economies which are part of
  the economic system
