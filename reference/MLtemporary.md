# Mean of the llk function used in the estimation of the selected ATSM

Mean of the llk function used in the estimation of the selected ATSM

## Usage

``` r
MLtemporary(xtemp, scaling_vector, MLvec)
```

## Arguments

- xtemp:

  temporary vector of parameters to be estimated numerically

- scaling_vector:

  scaling factor

- MLvec:

  log-likelihood function
