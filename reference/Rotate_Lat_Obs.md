# Rotate latent states to observed ones

Rotate latent states to observed ones

## Usage

``` r
Rotate_Lat_Obs(X_List, U1, U0)
```

## Arguments

- X_List:

  list of risk-neutral parameters: (i) intercept (A_X); slope (B_X) and
  volatility matrix (SS_X)

- U1:

  rotation matrix (N x N)

- U0:

  rotation vector (N x 1)
