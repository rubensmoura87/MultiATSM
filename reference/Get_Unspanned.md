# Collect both the domestic and global unspanned factors of all countries in single matrices

Collect both the domestic and global unspanned factors of all countries
in single matrices

## Usage

``` r
Get_Unspanned(ModelPara, FactorLabels, Economies, ModelType)
```

## Arguments

- ModelPara:

  List of model parameter estimates

- FactorLabels:

  A string-list based which contains all the labels of all the variables
  present in the model

- Economies:

  String-vector containing the names of the economies which are part of
  the economic system

- ModelType:

  A string-vector containing the label of the model to be estimated
