# Estimate K1h

Estimate K1h

## Usage

``` r
Est_K1h(horiz, min_mat, max_mat, time_step, B_coef)
```

## Arguments

- horiz:

  Minimum horizon observed

- min_mat:

  shortest maturity observed

- max_mat:

  longest maturity observed

- time_step:

  time unit of the model (scalar)

- B_coef:

  matrix of beta coefficients
